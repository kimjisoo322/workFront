객체

객체 선언

        - 중괄호{} 를 사용해서 객체를 생성하고 속성에는 모든 자료형이 올 수 있음
        - 객체는 속성명을 사용해서 속성을 식별

        * 속성명에 공백이나 특수문자 등이 포함된 경우 따옴표 '' 로 묶어서 표기 
          또한 객체에서 접근 시 무조건 대괄호를 사용해야 객체의 속성에 접근이 가능
         -> 특수문자, 공백이 포함된 경우 ★객체명['속성명']★ 으로 접근! 


        속성에 접근 : 객체명.속성명, 객체명[속성명]

객체의 메소드

        객체의 속성 중 함수 자료형인 속성 : 메소드

객체와 반복문

        객체가 가지고 있는 모든 속성들에 순차적으로 접근하려면
        단순 for문으로는 불가능. 따라서 for in 문을 사용해야 함.

객체의 속성 추가 및 제거

        객체를 생성한 이후에 새로운 속성을 추가하거나 제거할 수 있음
        객체.name = '이름';
        객체.name2 = '이름2'; 
        -> 새로 추가하면 갱신됨
        'name' in member 
        > 해당 속성이 정의되어있으면 true, 없으면 false 반환
        
        delete(member.name)
        -> 삭제가 완료되면 true 
        삭제 후 
        'name' in member 
        -> false 반환 (속성이 삭제되어 없어졌기때문!)
