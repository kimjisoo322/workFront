기본문법

1) 자료형

자바스크립트에서는 변수에 별도로 자료형(타입)을 지정하지 않음. 리터럴(입력한 값)에 의해 자료형이 결정됨 -> 변수에 값을 입력한 그 자체에 의해 자료형이 결정됨! 자바와의 큰 차이점!!!!

자료형을 확인하는 방법

typeof(값 또는 변수명)

Primitive type 원시타입

하나의 값만 저장하는 자료형
  • String 문자열

    • '', "" 로 묶은 데이터

  • number (숫자)

    • 정수와 실수 모두 포함

    숫자가 들어있어도 따옴표로 묶이면 문자열로 인식

  • boolean(논리값)

    • true, false

    falsy : false로 인정할 수 있는 값
    Boolean(0) : > fales로 반환 // boolean type으로 형변환
    0, '', "", `` 빈문자열, NaN, undefined(선언했을 때 초기값), null
    res = 0? 'true':'false'
    * undefined : 변수 선언만 하고 값을 할당하지 않았을 때 변수의 초기값
    * null : 값이 없거나 유효하지 않은 값 truthy : true로 인정할 수 있는 값 (falsy를 제외한 값)

객체

여러개의 원시타입을 하나로 묶어 놓은 것 ex) 배열, 함수 등
  • 배열
    •         1. 배열 정의하기 
          let 배열명 = [값, 값, 값, ... ]

          emptyArr = []; // 빈배열
          colors = ['red', 'blue'] //문자열배열
          arr = [10, 'banana'] // 여러 자료형을 함께 입력

        2. 배열 값에 접근하기 
         배열명[index]
         colors[0]; // red 
         colors.length; // 배열의 총길이
  • 함수
    •         1. 함수 정의하기 
        function 함수명( [매개변수1, 매개변수2, ... ]){
            return res;
        }
         - 매개변수 : 외부로부터 입력받는 값, 생략가능
         - return : 함수의 실행결과를 반환, 생략가능

        2. 함수 호출하기 
         함수명()
         함수명(인수, 인수, ...); 

        3. 기본 매개변수(ES6 추가)
         ->  함수에 인수값이 지정되지 않을 경우 기본매개변수값이 실행
        function 함수명(num1 = 0, num2 =0){
            return num1+num2;
        }
  • object 객체
    •         1. 객체 정의하기
            객체명 = {
                키:값,
                키:값,
                ...
            }
        2. 객체의 값에 접근하기 
            객체명.키이름
            객체명[키이름]

2. 데이터 형변환

문자열과 숫자의 '+' 연산

        문자열 + 숫자 = 문자열
        숫자 + 숫자 + 문자열 = 문자열
        (숫자가 먼저 계산되고 뒤의 문자열과 합쳐진다.)

        1) 문자열과 숫자의 '+' 연산
        문자
        (덧셈 이외의 연산자)
        숫자 * 문자열 = 숫자

2) 강제 형변환

내장 함수를 이용한 강제 형변환 
        Number(값), parseInt(값), parseFloat(값)
         문자형, 논리형 -> 숫자형 변환

        String(값), 값.toString()
         숫자형, 논리형 -> 문자형 변환

        Boolean(값)
         문자형, 숫자형 -> 논리형 변환

3. 연산자

'=='연산자와 '===' 연산자

'==' 은 값이 일치하는지 확인하는 연산자이다.
'===' 은 값과 자료형이 일치하는지를 확인하는 연산자이다.
