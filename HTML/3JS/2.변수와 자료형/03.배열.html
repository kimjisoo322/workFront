1. 배열

        자바스크립트에서 배열은 자료형을 지정하지 않기 때문에
        한 배열 안에 모든 자료형의 데이터 보관이 가능하다.

        배열 선언 시 크기를 지정하지 않고 선언할 수도 있고
        크기를 정하고 선언할 수도 있다.

2. 배열의 메소드

        배열도 하나의 객체이기 때문에 배열에서 활용할 수 있는 메소드들이 존재 

        
1) indexOf()

        : 배열에서 요소가 위치한 인덱스를 반환
        
        
        


        
2) concat()

        : 여러개의 배열을 결합하여 새로운 배열을 리턴
        
        


        
3) join()

        : 배열의 요소들을 결합해서 하나의 문자열을 반환
        
        


        
4) reverse()

        : 원본 배열의 순서를 뒤집는 메소드 
        
        



        
 5) sort()

        : 배열을 오름차순(문자기준)으로 정렬하는 메소드 
        
        

    
        
 6) push(), pop()

        ★ push() : 배열의 맨 뒤에 요소를 추가
        ★ pop () : 배열의 맨 뒤에 요소를 반환 후 제거

        
        


        
 7) shift(), unshift()

        ★ unshift () : 배열의 맨 앞에 요소를 추가
        ★ shift() : 배열의 맨 앞의 요소를 반환 후 제거

        
        


        
 8) slice(), splice()

        ★ slice () : 배열의 요소를 뽑아내는 메소드
        배열.slice(시작인덱스, 끝인덱스-1)
         -> 끝인덱스는 포함하지 않음 
         ex) slice(2,4)
         -> 2부터 3까지 반환
        ** 원본 배열에는 영향을 미치지 않음 > 추출해서 반환만!
        
        
        ★ splice() : 배열의 특정 인덱스 위치에 요소 제거 및 추가하는 메소드
        배열.splice(시작인덱스, 삭제될 개수, '추가할 값'); 
         -> 원본 배열에 영향을 미침
