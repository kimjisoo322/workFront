CSS 선택자 1

1. 모든 선택자(*)

HTML문서 안의 모든 태그를 선택할 때 전체 선택자 사용! 
 
        * {
            스타일 속성1:값1;
            스타일 속성2:값2;
        }

2. 태그 선택자(태그명)

태그명을 이용해 요소를 선택 :) 
        태그명 {
            스타일 속성1:값1;
            스타일 속성2:값2;
        }

3. id 선택자(#아이디명)

요소의 id속성을 이용해 요소 선택 :) 
        #id명{
            스타일 속성1:값1;
            스타일 속성2:값2;
        }

4. class 선택자(.클래스명)

HTML문서 내에 여러개의 태그를
동일한 클래스명으로 지정하여 선택할 때 사용! 
        .클래스명{
            스타일 속성1:값1;
            스타일 속성2:값2;
        }
