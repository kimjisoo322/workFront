CSS 고급선택자

1. 하위 선택자

부모 요소에 포함된 모든 하위 요소에 스타일을 적용
자식, 손자 모두 포함 
        1) 후손 선택자 (자식과 손자를 모두 포함)
            a b {
                스타일 속성 : 값;
            }
        2) 자식 선택자 
                        한단계 아래에 있는 자식만 선택하겠다     
            a > b {
                스타일 속성 : 값;
            }
            예를 들어, p 태그는 section 부모 안에 3개의 후손포함 자식이 있지만, section의 자식만 선택하고 싶어서 
            section > p 로 한단계 아래에 있는 자식 하나만 선택할 수 있다.

예약방법 & 사용요금

온라인 예약 신청

도미토리 : 25,000원/일

2. 동위 선택자

동위 관계(같은 레벨)에서 뒤에 위치한 태그를 선택할 때 사용 

        1) a 선택자 뒤에 있는 b 요소 한개를 선택 (인접 선택자)
        a + b {
            스타일 속성: 값;
        }
        >>  예) 선택한 것의 뒤에 있는 애만 스타일 적용됨 
        
        2) a 선택자 뒤에 있는 모든 요소를 선택
        a ~ b{
            스타일 속성: 값;
            스타일 속성: 값;
        } 예) 선택한 것의 뒤에 있는 애 부터 모두 스타일 적용됨

3. 속성 선택자

기본 선택자 뒤에 []를 사용해 속성과 속성값 사용 
        선택자[속성 = 속성값]{
            스타일 속성 : 값; 
            스타일 속성 : 값;
        }
